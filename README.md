# Cloud Cost Intelligence Platform

A cloud-native project to track AWS service costs and send alerts when spending exceeds a defined threshold.

## Objective
Build a real-world cloud system using AWS-native services, Infrastructure as Code, and containerization.

## Planned Stack
- AWS Lambda
- AWS Cost Explorer
- DynamoDB
- SNS
- Terraform
- Docker
